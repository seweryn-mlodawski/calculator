# Calculator

## Opis projektu

Prosty kalkulator wykonujący podstawowe operacje matematyczne: dodawanie, odejmowanie, mnożenie i dzielenie.

Program prosi użytkownika o:
1. Wybór operacji (1-4)
2. Podanie dwóch liczb
3. Wykonuje obliczenie i wyświetla wynik
4. Wszystkie zdarzenia są logowane do pliku `logfile.log`.

## Funkcjonalność

- Dodawanie, odejmowanie, mnożenie, dzielenie
- Walidacja danych wejściowych (sprawdzenie czy to liczby)
- Ochrona przed dzieleniem przez zero
- Logowanie wszystkich operacji
- Formatowanie wyniku (liczby całkowite i ułamkowe)

## Logowanie

Wszystkie operacje są zapisywane w pliku `logfile.log`:
- Wybrana operacja
- Argumenty obliczeń
- Wynik
- Błędy (nieprawidłowe dane, dzielenie przez zero)
